Website Reimagine

/ˈwebsīt iˈmajən/

Time: Winter 2020

Project Type: Personal

Tools: Figma

During the winter break, I decided to pick up a side project so that I can keep busy. I ended up creating a reimagine of my mother's Business Website as a mobile app using Figma.

There are two main features I found are the promotional aspect for the business and the ability to book appointments. Navigating towards the booking page is not difficult since it is constantly advertising the feature with highlighted buttons. Initially, there were no problems with the user experience with the website, so I did not necessarily improve anything. However, I enjoyed doing a rebranding for the website, learning Figma, and redesigning it as a mobile app.
