Random Colour Generator

/ˈrandəm ˈkələr ˈjenəˌrādər/

Time: Winter 2021

Project Type: Personal

Tools: Visual Studio Code

During this time, I wanted to learn how and understand a basic level of Javascript in order to do some front-end development. So, I thought that making a random colour generator would be a good start since I also genuinely wanted to understand how these things work. I enjoy figuring out things like this and the sense of pride I recieve from solving coding problems.

A notable problem I solved was trying to get the hexcode text to change with the colour in order for the user to identify and use it. Since the initial code for the generator edited the css document, I figured that it could also edit the html text. So, I first made the code identify the text I wanted to have edit, and then I replaced it with the same variable that the generator uses. Not the most complex problem, however I was proud that I understood the code and managed to come up with a solution.
